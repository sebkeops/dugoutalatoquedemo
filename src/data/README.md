# src/data

Données fictives **typées** du projet (TypeScript). Tout est mocké et présenté
comme une démonstration — aucune base de données réelle.

À peupler dans les phases suivantes :

- `univers.ts` — contenus de présentation (restaurant, traiteur).
- `soirees.ts` — soirées du mois.
- `evenements.ts` — 4 événements à emporter (Noël, Nouvel An, Saint-Valentin, Fête des mères).
- `messages.ts` — messages de contact / formulaire (mockup back-office).

Chaque entité pertinente porte un `minTier` (voir `src/tiers/types.ts`).
